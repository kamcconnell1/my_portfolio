JOBR

JOBR is an app to help users keep their job search organised. Users can upload potential jobs and assign them a status from wishlist through to rejected, add tasks to keep their ‘to-do’ list organised and upload resources they have found useful and contacts they want to keep. A full stack application built using Django and React.

Get-a-Gif

Get-a-Gif retrieves data from the Giphy API and allows users to search for a Gif, view the latest trending Gifs, add a Gif to their favourites to view later and copy a Gif to the clipboard. Produced over 2 days using React.
